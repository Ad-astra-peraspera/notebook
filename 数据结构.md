# 链表

> 链表是一种物理存储上非连续，数据元素的逻辑顺序通过链表中的指针链接次序，实现的一种线性存储结构。
>
> 链表由一系列节点（链表中每一个元素称为节点）组成，节点在运行时动态生成 （malloc），每个节点包括两个部分：
>
> + 存储数据元素的**数据域**
>
> + 存储下一个节点地址的**指针域**。最后一个节点的指针指向null（空指针）。头节点为head。
